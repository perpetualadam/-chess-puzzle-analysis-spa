# Chess App Icons

## How to Generate Icons

1. Open `generate-icons.html` in your browser
2. Icons will be automatically generated
3. Click the "Download" button under each icon size
4. Save all icons to this `/icons/` folder

## Required Icon Sizes

- **16x16** - Browser favicon
- **32x32** - Browser favicon
- **72x72** - Android Chrome
- **96x96** - Android Chrome
- **128x128** - Android Chrome
- **144x144** - Windows tile
- **152x152** - iOS Safari
- **192x192** - Android Chrome (standard)
- **384x384** - Android Chrome
- **512x512** - Android Chrome (high-res), iOS Safari

## Alternative: Use Online Icon Generator

If you prefer, you can use an online tool:

1. Go to https://realfavicongenerator.net/
2. Upload a 512x512 PNG image (chess knight on green background)
3. Download the generated icon pack
4. Extract all icons to this folder

## Icon Design

The default icon is a white chess knight silhouette on a green (#3fb959) background, matching the app's theme color.

You can customize the icon by:
1. Creating your own 512x512 PNG
2. Using the online generator above
3. Or modifying the `generate-icons.html` canvas drawing code
